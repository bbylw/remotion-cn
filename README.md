Remotion 是一个用于**使用 React 以编程方式创建视频**的框架。

## 为什么在 React 中创建视频？

- **利用 Web 技术**：使用所有 CSS、Canvas、SVG、WebGL 等。
- **利用编程**：使用变量、函数、API、数学和算法来创建新效果
- **利用 React**：可重用组件、强大的组合、快速刷新、包生态系统

## 开始使用

如果你已经安装了 Node.JS，输入

```console
npx create-video@latest
```

即可开始。否则，请阅读文档中的[安装页面](https://www.remotion.dev/docs/)。
